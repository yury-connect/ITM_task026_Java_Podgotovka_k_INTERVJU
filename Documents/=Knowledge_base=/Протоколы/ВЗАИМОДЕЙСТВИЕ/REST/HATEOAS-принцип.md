HATEOAS (*Hypermedia As The Engine Of Application State*) — это принцип архитектуры REST, при котором сервер вместе с данными возвращает клиенту динамические гипермедийные ссылки на следующие возможные действия, управляя состоянием приложения. [1](https://habr.com/ru/articles/483328/), [2](https://codechick.io/tutorials/rest-api-design/rest-hateoas), [3](https://en.wikipedia.org/wiki/HATEOAS), [4](https://egorzimowski.com/2022/09/06/rest-api-hateoas-in-java-spring-application-tutorial/)
### Как это работает
	
- Клиент получает от сервера не только чистый текст или числа, но и список ссылок.
	
- Ссылки показывают, какие действия доступны пользователю прямо сейчас.
	
- Клиент не должен сам угадывать или жестко прописывать адреса новых запросов в коде.

### Главные плюсы
	
- Гибкость: сервер может менять адреса путей без поломки клиентского приложения.
	
- Простая навигация: клиент исследует систему по ходу дела с помощью подсказок сервера.
	
- Полнота REST: это высший уровень зрелости API по модели Леонарда Ричардсона.

---
Пример (*Как выглядят ссылки в ответе*):
```json
{
  "id": 42,
  "status": "new",
  "total": 1990,
  "_links": {
    "self":   { "href": "/orders/42" },
    "pay":    { "href": "/orders/42/payment", "method": "POST" },
    "cancel": { "href": "/orders/42", "method": "DELETE" }
  }
}
```

## Форматы: HAL и JSON:API [ссылка](https://codechick.io/tutorials/rest-api-design/rest-hateoas)

Чтобы клиенты и серверы понимали друг друга одинаково, придумали стандартные форматы гипермедиа. Два самых известных:

|   |   |   |
|---|---|---|
|**Формат**|**Где ссылки**|**Особенность**|
|HAL|в ключе `_links`|минималистичный; вложенные ресурсы — в `_embedded`|
|JSON:API|в ключе `links` у ресурса и связей|строгая спецификация: `data`, `relationships`, `included`|

Пример HAL со встроенным ресурсом — заказ сразу содержит данные покупателя, чтобы клиент не делал второй запрос:
```json
{
  "id": 42,
  "total": 1990,
  "_links": { "self": { "href": "/orders/42" } },
  "_embedded": {
    "customer": {
      "id": 7,
      "name": "Анна",
      "_links": { "self": { "href": "/users/7" } }
    }
  }
}
```

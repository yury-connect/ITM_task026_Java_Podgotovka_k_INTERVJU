# Задачка: 
### Дана таблица `orders(id, customer_id`). Нужно получить список `customer_id`, у которых <u>2 и более заказа</u>.

---
Для решения этой задачи нужно сгруппировать записи по `customer_id`, посчитать количество заказов для каждого клиента и оставить только тех, у кого это количество **2 или более**.

Вот универсальное решение, которое работает во всех основных СУБД:

### Используем `GROUP BY` и `HAVING`
```sql
SELECT customer_id
FROM orders
GROUP BY customer_id
HAVING COUNT(id) >= 2;
```

### Пояснение:
- **`GROUP BY customer_id`** — группируем заказы по каждому клиенту    
- **`COUNT(id)`** — считаем количество заказов у каждого клиента    
- **`HAVING COUNT(id) >= 2`** — оставляем только те группы (клиентов), у которых заказов 2 или больше    

### Результат:
Список `customer_id`, у которых минимум 2 заказа. Если у клиента 1 заказ или их нет — он не попадет в выборку.

### Если нужно вывести еще и количество заказов:
```sql
SELECT 
    customer_id, 
    COUNT(id) AS orders_count
FROM orders
GROUP BY customer_id
HAVING COUNT(id) >= 2
ORDER BY orders_count DESC;  -- можно отсортировать по убыванию
```

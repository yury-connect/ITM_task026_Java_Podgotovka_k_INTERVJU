# Задание_№1

```java
public class Main {
    public static void main(String[] args) {
        A objA = new A();
        B objB = new B();
		
        objA.b = objB;
        objB.a = objA;
		
        // Удалятся ли объекты A и B?
        System.gc();
    }
}
```

---
**Нет, не удалятся.**

**Причина:**  
Объекты `A` и `B` имеют ссылки друг на друга (`objA.b = objB` и `objB.a = objA`). Это создает **циклическую ссылку**.
- **Сборщик мусора в Java (современные версии)** использует алгоритмы, которые проверяют **достижимость** объектов от **GC Roots** (корневых точек).    
- `objA` и `objB` — это локальные переменные в стеке (`main` метод). Пока метод `main` выполняется, эти переменные являются **GC Roots**.    
- Через переменные `objA` и `objB` объекты **достижимы**, следовательно — они **живые**.

**Вывод:**  
Объекты удалятся только **после** того, как метод `main` закончится, и переменные `objA` и `objB` выйдут из области видимости (или им будет присвоено `null`).

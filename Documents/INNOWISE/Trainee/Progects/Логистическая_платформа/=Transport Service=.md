## **Transport Service**
### Задачи:
- реестр водителей;
- назначение водителей на транспортные средства (ТС); 
- реестр ТС;
- графики ТО TC;
### Особенности:
- Высокая нагрузка (работа с множеством заказчиков);
- Масштабируемость / Лекговесность;
- Фиксирует усредненное местоположение (т.е. только город, только для заказчиков)
- Если происходит ЧП с транспортом - то уведомляет сервис доставки;
- Все протоколирует в Reporting Service;
### Стек:
- DB: Postgres;
- Kafka;

---

### Транспортные средства (ТС):
- Добавить новый ТС в автопарк предприятия;
- Получить информацию о ТС/ получить все ТС с пагинацией и без;
- Изменить статус ТС;
  - по занятости (занят в перевозке/ свободен для приема доставки)
  - по тех. состоянию (исправен/ неисправен/ на обслуживании/ в ремонте)
- Удалить ТС из автопарка;

### Водители
- Добавить нового водителя
- Получить информацию о водителе/ всех водителей с пагинауией и без;
- Изменить:
  - Прикрепить/ открепить ТС к данному водителю (у водителя может быть несколько ТС);
  - Статус водителя (работоспособен/ болен-на больничном/ на выходном/ временно неработоспособен)
- Удалить водителя;

### График Тех. обслуживания ТС
- назначить на какой-то день

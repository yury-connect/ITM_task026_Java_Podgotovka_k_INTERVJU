## **Paymant Service**
### Задачи:
- Рассчитывает и формирует стоимость доставки груза;
- Актуализирует из внешних источников инфу для расчета стоимости доставки;
- Контролирует оплату пользователя за доставку;
### Стек:
- DB: Postgres;

---
### **Платежи** (Paymants):

| Метод | Эндпоинт                     | Описание (кратко)                           | Бизнес-правила                                                                                        |
| ----- | ---------------------------- | ------------------------------------------- | ----------------------------------------------------------------------------------------------------- |
| GET   | `/api/v1/paymants/{id}/cost` | Рассчитать стоимость *(просто узнать цену)* | с ролью SERVICE Базовая_ставка + вес*10 + расстояние*5                                                |
| POST  | `/api/v1/paymants/bill`      | Сформировать счет на оплату                 | синхронно идет за расчетами в Paymant Service  с ролью SERVICE Базовая_ставка + вес*10 + расстояние*5 |

Формирование счета:
1. Исходя из `Базоваой_ставки`, `веса` груза и `расстояния` формируется стоимость доставки `cost`;
2. Вычисленная величина `cost` записывается в БД и ей присваивается уникальный `id` с типом `UUID`;
3. Клиенту возвращается счет для оплаты `bill`, который включает `сумму` для оплаты,  `id` платежа и срок действия счета `durationDays` в календарных днях;
4. Далее `Delivery Service` бронирует одно из имеющихся в автопарке транспортных средств на указанное в счете время. По истечении указанного интервала `durationDays`;

| Метод | Эндпоинт                     | Описание                                    | Бизнес-правила                                         |
| ----- | ---------------------------- | ------------------------------------------- | ------------------------------------------------------ |
| GET   | `/api/v1/paymants/{id}/cost` | Рассчитать стоимость *(просто узнать цену)* | с ролью SERVICE Базовая_ставка + вес*10 + расстояние*5 |

#### bill

| имя переменной | тип        | Пояснение                                              |
| -------------- | ---------- | ------------------------------------------------------ |
| cost           | BigDecimal | Стоимость хранится в минимальных единицах (копейках)   |
| durationDays   | int        | Когда текуща дата станет больше даты выписывания счета |

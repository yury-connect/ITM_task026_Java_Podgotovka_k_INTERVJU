- **API Gateway**: Единая точка входа для мобильных приложений (для водителей) и веб-интерфейсов.
  
- **Event-Driven Architecture (EDA)**: Использование брокеров сообщений - **Apache Kafka**) для обмена данными между сервисами (обновление статуса доставки мгновенно уведомляет сервис биллинга).
  
- **Saga Pattern**: Обеспечение целостности данных при выполнении распределенных транзакций (например, бронирование груза на нескольких этапах перевозки).

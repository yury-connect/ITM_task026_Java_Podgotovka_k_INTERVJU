# Какие бывают модификаторы?

_**🔘**_ Модификаторы доступа `private`, `protected`, `public` (рассмотрим подробнее в разделе #Классы)  
_**🔘**_ Модификаторы для многопоточности `synchronized` и `volatile` (подробнее чуть позже)  
_**🔘**_ `static` (рассмотрим подробнее в разделе #Классы)  
_**🔘**_ [final](https://itsobes.com/ru/java/chem-otlichaetsia-final-finally-finalize)  
_**🔘**_ `abstract` (рассмотрим подробнее в разделе #Классы)  
_**🔘**_ `native` – реализация метода скрыта внутри JVM, нельзя указывать в пользовательском коде  
_**🔘**_ `transient` – поле будет пропущено при сериализации  
_**🔘**_ `strictfp` – самый экзотический, ограничивает точность вычисления для переменной с плавающей точкой до стандарта IEEE. [Нужно для переносимости между платформами](https://stackoverflow.com/q/517915/5521491).

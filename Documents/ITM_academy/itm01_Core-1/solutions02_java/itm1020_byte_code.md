# Что такое byte code?

**Байт-код Java** – это промежуточный набор инструкций, скомпилированный из исходного кода Java, который выполняется **JVM**.

Он делает Java кроссплатформенной, так как может исполняться на любой системе с установленной виртуальной машиной.

```
***** из методички *****
Байт-код Java — набор инструкций, скомпилированный компилятором, исполняемый JVM. 
```

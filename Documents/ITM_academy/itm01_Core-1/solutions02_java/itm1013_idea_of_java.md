# Какая основная идея языка?

Основная идея Java – **"Написано однажды – работает везде"**. Это означает, что код, написанный на Java, может выполняться **на любой платформе без изменений**, благодаря использованию виртуальной машины _(JVM)_.

```
***** из методички *****
"«Написано однажды - работает везде».
Идея основывается в написании одного кода, который будет работать на любой платформе."
```

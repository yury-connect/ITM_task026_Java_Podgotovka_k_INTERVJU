# Какие недостатки у java?

**Недостатки Java**:

> **Платное коммерческое использование** (с 2019 года) – лицензирование требует затрат для некоторых видов использования.
> 
> **Низкая производительность** – из-за использования JVM и сборщика мусора, что может замедлить выполнение приложений.
> 
> **Неудобные инструменты для GUI** – создание графических интерфейсов на чистой Java ограничено.
> 
> **Многословность кода** – язык требует большего количества кода для выполнения операций по сравнению с более компактными языками.
